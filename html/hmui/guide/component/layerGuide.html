레이어팝업

레이어팝업 공통(B-Pop/F-Pop/Confirm)

fnOpenPop 클래스로 팝업 오픈할 시 data-name="오픈할 팝업 ID" 필수

- 팝업오픈 함수 호출 시 : fnOpenLayerPop('오픈할 팝업 ID')

- 팝업 오픈 후 인터렉션 실행 필요 시 exeTransitionInLayer에서 정의

fnClosePop 클래스가 있는 경우 해당 팝업을 닫고 data-name="닫은 팝업ID"로 포커스 이동

- 팝업닫기 함수 호출 시 : fnCloseLayerPop('닫을 팝업 ID', '포커스 보낼 element')

B-POP

- 네이티브 dim을 사용하지 않는 바텀시트는 data-dim="true" 속성 추가 (네이티브 dim과 중복으로 케이스 분리)

- 타이틀이 영역보다 클경우 font-size resize (fnOpenLayerPop 시 체크)

<!-- 콘텐츠 내 팝업 호출 버튼 --> <button type="button" class="btn fnOpenPop" data-name="popupId">버튼</button> <!-- popup --> <div id="popupId" class="wrap_layer" data-dim="true"> <div class="inner_layer"> <div class="head_layer"> <h2 class="tit_layer">타이틀</h2> </div> <div class="content_layer"> // popup contents here </div> <div class="section_bottom_fixed"> <div class="wrap_btn_full"> <button type="button" class="btn_full_primary fnClosePop">하단고정버튼</button> </div> </div> <button class="btn_ico_close fnClosePop"><span class="blind">닫기</span></button> </div> </div> <!-- // popup -->

F-POP

- 타이틀이 영역보다 클경우 font-size resize (fnOpenLayerPop 시 체크)

<!-- 콘텐츠 내 팝업 호출 버튼 --> <button type="button" class="btn fnOpenPop" data-name="popupfullId">버튼</button> <!-- popup --> <div id="popupfullId" class="wrap_layer type_full"> <div class="inner_layer"> <div class="head_layer"> <h2 class="tit_layer">타이틀</h2> </div> <div class="content_layer"> // popup contents here </div> <div class="section_bottom_fixed"> <div class="wrap_btn_full"> <button type="button" class="btn_full_primary">하단고정버튼</button> </div> </div> <button class="btn_ico_close fnClosePop"><span class="blind">닫기</span></button> </div> </div> <!-- // popup -->

Confirm

- 컨펌창은 dim영역 클릭시 닫는 기능 없음

<!-- 콘텐츠 내 팝업 호출 버튼 --> <button type="button" class="btn fnOpenPop" data-name="confirmId">버튼</button> <!-- popup --> <div id="confirmId" class="wrap_layer type_alert" data-dim="true"> <div class="inner_layer"> <div class="content_layer"> <p class="txt_alert">confirm msg here</p> </div> <div class="section_button_alert"> <div class="wrap_btn_full"> <button type="button" class="btn_full_secondary fnClosePop">button</button> <button type="button" class="btn_full_primary fnClosePop">button</button> </div> </div> </div> </div> <!-- // popup -->

Toast

- fnToastPop('toast팝업 ID') 호출 시 토스트팝업 노출 후 사라짐

<!-- toast --> <div id="toast" class="wrap_toast"> <p class="txt_toast">토스트팝업 메시지</p> </div> <!-- // toast -->

Tooltip

- fnOpenTooltip 클래스 data-name="오픈할 툴팁ID" 필수

- 툴팁위치에 따라 left / center / right 계산하여 위치 조정

- 툴팁이 열려 있는 경우 툴팁 외의 영역 터치 시 툴팁 닫음

<!-- 콘텐츠 내 툴팁 호출 버튼 --> <button type="button" class="btn_tooltip fnOpenTooltip" data-name="tooltip"> <span class="ico_tooltip"><span class="blind">툴팁</span></span> </button> <!-- tooltip --> <div id="tooltip" class="wrap_tooltip"> <div class="inner_tooltip"> <p class="tit_tooltip">툴팁타이틀</p> <p class="txt_tooltip">툴팁 내용</p> <button class="btn_ico_close fnCloseTooltip"><span class="blind">닫기</span></button> </div> </div> <!-- // tooltip -->
