  • 체크카드

체크카드

  1. ATM 기기 위치 검색
    • 마스터카드 ATM
    • 유니온페이 ATM
  2. 비밀번호 입력

    PIN번호 4자리 요청 시
    카드 비밀번호 4자리

    PIN번호 6자리 요청 시
    카드 비밀번호 4자리 + 00(숫자)

  3. 인출방법 선택

    출금(Withdrawal) 혹은 잔액(Balance) 버튼 터치

  4. 출금 계좌 유형 선택

    보통예금(Savings 혹은 Checking account) 버튼 터치

  5. 출금 금액 입력

    ATM 인출 한도
    일 USD 6,000 / 월 USD 10,000
    (기기마다 1회 인출 가능 금액 다름)

  6. 출금 통화 선택

    현지 통화 선택
    ex. 일본에서 출금 시 JPY 선택

    USD Conversion, 원화(KRW)를 선택하면 안돼요

    수수료가 부과되거나 거래가 거절돼요.

  7. 인출결과 확인 후 수령하면 끝!

ATM 기기마다 진행 순서가 다를 수 있어요.

현금 인출 후 카드를 꼭 챙겨주세요!

신용카드

  1. 해외결제 설정 확인

    충전한 외화 하나머니 인출 시
    ‘외화 하나머니’로 설정

    해외결제 설정하러 가기
  2. ATM 기기 위치 검색
    • 마스터카드 ATM
  3. 비밀번호 입력

    PIN번호 4자리 요청 시
    카드 비밀번호 4자리

    주변의 은행 ATM 기기를 이용해 주세요.
